Add support for special content types in your website, such as a portfolio, client, and team member.





== Description ==

**NOTE: This plugin is meant for use with the WordPress themes developed by [Imon Themes](http://imonthemes.com) theme Safreen, which take advantage of it to add richer content areas and designs. Check them out!** 

[Safree Widgets](http://imonthemes.com/safreen) is a utility plugin that adds support for a specific set of content elements within your WordPress installation. This plugin will add 3 custom post types to your site:team members, clients,about us . You can still use safreen widgets for any WordPress theme, although you will have to create your own page templates.


= Included Content Types =
* Team Members
* About Us
* Clients

== Installation ==

= Installing Through The WordPress Admin =

1. Download the ZIP file
2. In your WordPress admin area, go to Plugins > Add Plugin, and select Upload Plugin
3. Upload the ZIP file and installation will commence
4. Activate the plugin through the 'Plugins' menu in WordPress

= Installing Through FTP =

1. Download the ZIP file and unpack it
2. Upload the entire **Safreen widgets** to the **/wp-content/plugins/** directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Make sure to check the **Settings > Safreen widgets** page before using it right away!
